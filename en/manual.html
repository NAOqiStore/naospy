NAO Spy Manual

    Download NAOSpyServer

    NAOSpyServer is a NAO robot application,you need to download and install the application on your robot.
    Download: https://naoqistore.github.io/naospy/en/download.html

    Install NAOSpyServer

    • Use Choregraphe to open NAOSpyServer
    • Click on the upload button to upload the application to the robot
    • Click the run button to run the NAOSpyServer behavior
    • Suggestion：Set the NAOSpyServer as default behavior so it can be automatically executed at robot startup
    • Choregraphe details of the use of instructions

    Connection

    • Make sure the NAOSpyServer behavior is running on your robot
    • Make sure your iPhone or iPad and your NAO robots are Connected to the same router.Do not use the company or the school's public network
    • Open the NAO Spy app, select a robot in the list of robots, click the "Connect" button
    • If there are no robots in the list, try entering the IP address of the robot directly
